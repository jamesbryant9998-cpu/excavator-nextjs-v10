# Komatsu PC220 Excavator Product Information

## Product Overview
The Komatsu PC220 is a high-performance hydraulic crawler excavator designed for a wide range of applications, including construction, mining, and roadwork. It is known for its high efficiency, durability, and low operational costs.

## Technical Specifications
| Parameter | Value |
| --- | --- |
| **Model** | Komatsu PC220 Excavator |
| **Operating Weight** | 22,000 kg (22 Tons) |
| **Bucket Capacity** | 1.0 m³ |
| **Engine Model** | Komatsu SAA6D107E-1 |
| **Engine Power** | 125 kW / 168 HP |
| **Max Digging Depth** | 6,620 mm |
| **Max Digging Reach** | 9,875 mm |
| **Max Digging Height** | 10,000 mm |
| **Undercarriage Condition** | 95% New |
| **Fuel Type** | Diesel |

## Key Selling Points
- **Outstanding Efficiency**: Advanced hydraulic system ensures faster cycle times and superior digging force.
- **Engine Reliability**: Equipped with a genuine Komatsu SAA6D107E-1 engine for powerful and stable performance.
- **Built to Last**: Reinforced boom, arm, and high-strength frame provide long-term reliability in tough working environments.
- **Fuel Economy**: Intelligent engine management and hydraulic control reduce fuel consumption without sacrificing power.
- **Operator Comfort**: Spacious, ergonomic cabin with low vibration and noise levels for better operator productivity.
- **Easy Serviceability**: Simplified access to engine and hydraulic filters for quick routine maintenance.
